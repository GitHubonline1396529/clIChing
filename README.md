# clIChing: 基于命令行交互的周易卦盘

## 项目简介

脑洞大开搞出来的很抽象的项目，在命令行里面用《周易》(I Ching) 赛博算卦。使用的语言是 Python，目前刚刚做到了能跑起来并输出卦象的程度，连卦辞都只是随便在网上找的。结果做到一半发现项目查重没过，好家伙用类似的项目在 [GitHub 上面一抓一大把](https://github.com/search?q=I+Ching)，甚至 [专门的包](https://pypi.org/project/iching/)[^1] 都有，真是离大谱了。

但是好处是这个项目还是有点优势的，比如：

1. 是交互式的命令行程序，这也就是说如果你想浏览你的卦的所有爻变还是蛮方便的。
2. 会帮你用 `#` 画一个卦象，而且输出是彩色的[^2]，标注出所有可能的变爻。
3. 在未来的一个月内我还有更新它的动力。

那反正还是先上传再说吧，后面再逐步完善。

## 使用方法

目前除了代码注释之外还没有任何的文档，而且我还没进行可执行文件的封装呢。不过如果你想运行这个项目可以直接使用下面的命令：

```bash
python -u main.py 
python -u main.py --help  # 查看帮助信息
```

进入程序后会显示 `divination table >>>` 字样的 Prompt，你可以执行如下的输入：

- `q` / `quit` / `exit` / `system` - 退出卦盘，回到终端命令行；
- `g` / `get` - 获取一个本卦，重复输入将会覆盖原卦。
- `c` / `change` - 使用形如 `c 1 3 4` 或者 `change 1,2,6` 这样的输入获取本卦的变卦，其中的数字指定发生爻变的爻 (前提是这些爻都必须是老爻)。
- `h` / `help` - 显示帮助信息。
- `clear` / `reset` - 清除当前的卦象。
